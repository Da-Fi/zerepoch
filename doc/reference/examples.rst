Examples
========

Full examples of Zerepoch Applications can be found in the ``zerepoch`` `repository <https://github.com/The-Blockchain-Company/zerepoch/tree/master/zerepoch-use-cases>`_.
The source code can be found in the ``src`` and the tests in the ``test`` folder.

The examples are a mixture of simple examples and more complex ones, including:

- A crowdfunding application
- A futures application
- A stablecoin
- A uniswap clone

.. important::
  Make sure to look at the same version of the `zerepoch` repository as you are using, to ensure that the examples work.
