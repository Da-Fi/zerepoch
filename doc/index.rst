The Zerepoch Platform and Simeon
===============================

The Zerepoch Platform is an application development platform for the Bcc blockchain.
To read more about the Platform, see :ref:`what_is_the_zerepoch_platform`.
To get started using the Platform see :ref:`zerepoch_getting_started`.

Simeon is the flagship product built on top of the Zerepoch Platform, consisting of a domain-specific language for writing financial smart contracts.
To read more about Simeon, see the :ref:`Simeon tutorials<simeon_tutorials>`.
